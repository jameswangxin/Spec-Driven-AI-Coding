# CAP-0001：报表 CSV 导出

## 能力边界

### 覆盖范围

- 将报表查询结果导出为 CSV。
- 复用报表权限校验。

### 不覆盖范围

- Excel 导出。
- 异步导出。
- 邮件发送。

## 使用方式

后续报表导出类需求可以引用：

```yaml
capabilities:
  - CAP-0001
```

## 依赖关系

- 相关需求：`REQ-0001`。

## 约束

- 导出必须经过权限校验。
- 导出字段不得绕过脱敏规则。

## 示例引用

### 适合引用的场景

- 增加新的 CSV 报表导出。

### 不适合引用的场景

- 异步大文件导出。
