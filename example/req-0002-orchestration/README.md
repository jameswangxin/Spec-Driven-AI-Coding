# REQ-0002 示例：Skill 编排与审计

本目录展示一个目标项目如何记录「为 workflow Skills 增加可审计自动编排能力」的需求链路。

示例文件：

```text
REQ-0002.md                  # 需求工作单
REQ-0002-plan.md             # 技术方案
REQ-0002-implementation.md   # 实施记录（待后续补充）
```

这些文件用于参考写法，不要直接复制到目标项目作为真实需求。

## 与本项目的关系

- 本仓库根目录的 `.workflow/` 记录模板项目自身需求（如 REQ-0001）。
- `example/req-0002-orchestration/` 是一个独立示例，演示如何以企业级、可审计的方式记录 Skill 编排需求。
